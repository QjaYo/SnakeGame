# SnakeGame

2024/08/24

### 제작계기
학교에서 C는 배웠으나
큐, 스택, 벡터 등 라이브러리들이 없어 매번 만들어야하는게 불편하였고,
객체지향프로그래밍도 배워보고 싶어
SnakeGame 만드는 것을 목표로 C++를 학습했다.

### 설명
읽기 쉽고, 보수가 쉬운 코드를 만들어보려 노력했다.
(enum을 이용해 숫자에 의미부여, 반복적으로 쓰이는 값은 변수에 담아 하드코딩을 피함)
C++가 제공해주는 기능들을 최대한 많이 활용해보고자 노력했다.
Windows API를 처음으로 사용해보았다.
fileStream헤더를 이용해 txt 파일로 사용자의 최고기록 리스트를 저장하고 불러올 수 있게 구현했다.

### 실행
exe파일을 실행하면 플레이할 수 있다. (windows)
